BankSlip Keylogger Malware Analysis – CyberShield Solutions Simulation

Scenario: Malware Analyst Internship at CyberShield Solutions
Role: Malware Analysis Intern, Threat Intelligence Team
Focus: Static & Dynamic Analysis, IOC Extraction, Reporting
Collaboration: Conducted jointly with a project partner

As part of a simulated internship with CyberShield Solutions’ Cybersecurity Division, I acted as a Malware Analysis Intern investigating threats impacting fictional client environments. This professional scenario replicated a real-world malware analysis assignment, where I was tasked with analyzing two guided malware samples followed by an independently selected one — the BankSlip Keylogger — to apply my workflow and reporting process.

The BankSlip Keylogger investigation involved both static and dynamic analysis using tools such as IDA, PEstudio, RegShot, Procmon, and Volatility. The analysis revealed keystroke capture functions, registry persistence behavior, and network communication through API calls such as InternetConnectW and HttpSendRequestW. This project followed CyberShield’s structured malware analysis workflow, including file hashing, sandbox execution, process monitoring, and documentation of Indicators of Compromise (IOCs).
