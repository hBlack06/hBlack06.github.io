Computer Science Course List

Course Code Course Name Semester Year Description
CSC-1710 Intro. to Computer Programming Fall 2022 An introduction to the fundamentals of programming using a high-level, structured programming language such as C. Topics include problem-solving methods, algorithms, program design, coding, debugging, testing, and documentation. Students will develop programs emphasizing good coding style, structured programming techniques, and logic.
CSC-2342 Discrete Structures Spring 2023 This is an introductory course in discrete mathematical structures widely used in computer science. Topics include logic, sets, functions, relations, combinatorics, graph theory, and proofs. Applications of these topics to computer science areas such as algorithms and data structures are emphasized.
CSC-1720 Adv Prog With Data Structures Spring 2023 A continuation of CSC 1710 covering more advanced programming fundamentals including recursion, sorting, searching, dynamic memory allocation, and the implementation of abstract data types such as stacks, queues, and trees. Emphasis is placed on efficient data organization and algorithmic problem solving.
CSC-2710 Adv Data Structures/Algo Dev Fall 2023 Focuses on algorithm development using advanced data structures such as heaps, graphs, and balanced trees. Topics include algorithm complexity analysis, recursion, searching, sorting, and graph algorithms. Emphasis on developing efficient, maintainable, and scalable software solutions.
CSC-2410 Computer Systems Fall 2023 Introduction to instruction set architecture, memory organization, assembly language, and how hardware and software interact within a computer system. Topics include binary representation, CPU organization, and the role of compilers and operating systems in system performance.
CSC-3810 Intro to Information Security Spring 2024 Introduces key information security concepts including confidentiality, integrity, authentication, risk management, and threat modeling. Students gain hands-on experience with encryption, access control, and system hardening through practical labs and simulations.
CSC-2212 Database Systems Spring 2024 Covers database design principles, entity-relationship modeling, SQL programming, relational algebra, and normalization. Students will design and implement databases, explore transaction management, and gain exposure to NoSQL and distributed database systems.
CSC-3820 Penetration Testing Fall 2024 Explores penetration testing methodology, vulnerability discovery, exploitation techniques, and ethical hacking practices. Students learn how to perform security assessments and create professional penetration testing reports while emphasizing responsible disclosure and communication with stakeholders.
CSC-3940 System Security Fall 2024 Focuses on systems thinking, vulnerability discovery, system hardening, and secure administration. Topics include access control, authentication mechanisms, system monitoring, and patch management. Students learn to secure real-world operating systems and networked environments.
CSC-4710 Software Engineering Spring 2025 Capstone course covering software design principles, object-oriented programming (OOP), agile methodologies, testing, and version control. Students work in teams to plan, design, implement, and present large-scale software development projects using professional tools and techniques.
CSC-4210 Operating Systems Spring 2025 Explores core operating system concepts such as process scheduling, memory management, file systems, and synchronization. Students complete hands-on programming projects that simulate or modify kernel-level behavior in UNIX-like environments.
CSC-4820 Reverse Engin & Malware Analys Fall 2025 Teaches reverse engineering, static and dynamic binary analysis, and malware investigation using real-world industry tools. Students learn to analyze malicious code behavior, understand obfuscation techniques, and create detailed technical reports.
CSC-3460 Networking/Network Programming Fall 2025 Introduces fundamental networking concepts including the OSI and TCP/IP models, IP addressing, routing, and sockets programming. Students build client-server applications and analyze network performance through practical, project-based exercises.
MTH-3010 Cryptography Spring 2025 This course is an introduction to the mathematical foundations of cryptography and cryptanalysis. Topics include elementary number theory, classical and modern symmetric key cryptosystems, public key cryptography, primality tests, factoring algorithms, hash functions, and digital signatures. Selected further topics may include security protocols, digital cash, elliptic curve cryptography, or quantum cryptography. Prerequisites: CSC 1710, MTH 2210 or CSC 2342. Four credits.
