Computer Science Course List

Course Code Course Name Semester Year Description
CSC-1710 Intro. to Computer Programming Fall 2022 An introduction to the fundamentals of programming using a high-level, structured programming language such as C. Topics include problem-solving methods, algorithms, program design, coding, debugging, testing, and documentation. Students will develop programs emphasizing good coding style, structured programming techniques, and logic.
CSC-2342 Discrete Structures Spring 2023 Introductory course in discrete mathematical structures widely used in computer science. Topics include logic, sets, functions, relations, combinatorics, graph theory, and proofs, with applications to algorithms and data structures.
CSC-1720 Adv Prog With Data Structures Spring 2023 A continuation of CSC 1710 covering recursion, sorting, searching, dynamic memory allocation, and implementation of abstract data types such as stacks, queues, and trees, with emphasis on efficient data organization.
CSC-2710 Adv Data Structures/Algo Dev Fall 2023 Focuses on algorithm development using advanced data structures such as heaps, graphs, and balanced trees, along with complexity analysis, recursion, searching, sorting, and graph algorithms.
CSC-2410 Computer Systems Fall 2023 Introduction to instruction set architecture, memory organization, assembly language, and how hardware and software interact within a computer system, including binary representation and CPU organization.
CSC-3810 Intro to Information Security Spring 2024 Introduces key information security concepts including confidentiality, integrity, authentication, risk management, and threat modeling, with hands-on labs in encryption, access control, and system hardening.
CSC-2212 Database Systems Spring 2024 Covers database design principles, entity-relationship modeling, SQL, relational algebra, and normalization. Students design and implement databases and explore transaction management and distributed systems.
CSC-3820 Penetration Testing Fall 2024 Explores penetration testing methodology, vulnerability discovery, exploitation techniques, and ethical hacking practices. Emphasis on security assessments and professional reporting.
CSC-3940 System Security Fall 2024 Focuses on systems thinking, vulnerability discovery, system hardening, access control, authentication mechanisms, monitoring, and patch management to secure real-world operating systems and networked environments.
CSC-4710 Software Engineering Spring 2025 Capstone course on software design principles, OOP, agile methodologies, testing, and version control. Students work in teams to plan, design, implement, and present large-scale software projects.
CSC-4210 Operating Systems Spring 2025 Explores core OS concepts such as process scheduling, memory management, file systems, and synchronization. Includes hands-on programming projects that modify or simulate kernel-level behavior.
CSC-4820 Reverse Engin & Malware Analys Fall 2025 Teaches reverse engineering, static and dynamic binary analysis, and malware investigation using real-world tools. Students analyze malicious code, study obfuscation techniques, and generate detailed technical reports.
CSC-3460 Networking/Network Programming Fall 2025 Introduces networking concepts including OSI and TCP/IP models, IP addressing, routing, and sockets programming. Students build client-server applications and analyze network performance through projects.
MTH-3010 Cryptography Spring 2025 Introduction to the mathematical foundations of cryptography and cryptanalysis, including number theory, symmetric and public key systems, hash functions, digital signatures, and potentially elliptic curves or quantum cryptography.
